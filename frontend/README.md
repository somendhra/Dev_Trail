# AI-Powered Parametric Insurance Frontend

This is a Vite + React + Tailwind frontend scaffold for the "AI Insurance for Gig Workers" platform.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

Routes available:
- `/` Landing
- `/register` Register
- `/login` Login
- `/dashboard` Dashboard
- `/plans` Plans
- `/payment` Payment
- `/claims` Claims
- `/notifications` Notifications
- `/chat` Chat Support
- `/profile` Profile
